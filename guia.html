Manual de Uso Exhaustivo: Salguacate ERP

Sistema Integrado de Gestión Comercial y Despliegue en la Nube

Bienvenido al manual oficial de Salguacate ERP. Esta herramienta no solo digitaliza el día a día del negocio, sino que está preparada para un despliegue gratuito en la nube (Render) y su distribución a través de dispositivos móviles Android. A continuación se detallan todas las operativas tecnológicas y de uso de la aplicación.

1. Despliegue e Infraestructura Tecnológica

1.1 Despliegue en Render (Cloud)

El backend de la aplicación está diseñado para alojarse de manera gratuita y automática en Render:

  • Suba el repositorio de la aplicación a GitHub.
  • En Render (render.com), cree un "New Web Service" y vincule su repositorio de GitHub.
  • Render leerá automáticamente el archivo render.yaml incluido en la raíz del proyecto y configurará el entorno de Node.js.

1.2 Conexión con Base de Datos Distribuida (Turso/libSQL)

Para evitar que el proveedor de hosting gratuito elimine su base de datos (problema común con SQLite), la aplicación tiene integración nativa con Turso (hasta 9GB gratuitos en la nube):

  • Inicie sesión en Turso y cree una base de datos.
  • Vaya a la configuración de variables de entorno de su Web Service en Render y añada: TURSO_DATABASE_URL y TURSO_AUTH_TOKEN.
  • El código detectará la configuración de Turso de forma automática sin que tenga que modificar la base de código. Si no se proveen estas variables, la aplicación usará SQLite de manera local (ideal para desarrollo).

1.3 Aplicación Nativa para Android

Se incluye un proyecto nativo completo en la carpeta /android:

  • Para generar la aplicación, primero ejecute el comando npm run build:android desde la terminal. Esto empaquetará el entorno React e inyectará los recursos dentro de la carpeta local de la app de Android.
  • Abra Android Studio, cargue la carpeta /android y seleccione Build > Build APK. Este APK se puede instalar en cualquier tableta del local.

2. Acceso al Sistema

El sistema cuenta con un acceso optimizado para pantallas táctiles (TPVs y Tablets). No es necesario recordar nombres de usuario complejos. Cada empleado tiene su perfil visual.

  • Pulse sobre su Nombre / Perfil en la lista.
  • Introduzca su PIN de 4 dígitos usando el teclado numérico.
Seguridad: El sistema cerrará la sesión automáticamente al recargar la página para prevenir el acceso no autorizado en terminales compartidos.

3. Panel de Control Principal (Dashboard)

El Dashboard es el centro neurálgico para los perfiles de Administración (Manager y Owner). Ofrece una vista panorámica en tiempo real del estado del negocio.

  • KPIs Superiores: Visualice de un vistazo las ventas del día, los gastos registrados y el número de incidencias.
  • Alertas de Stock: Si el stock de cualquier producto crítico baja de su límite mínimo, aparecerá una alerta inmediata.
  • Presencia en Tiempo Real: Muestra quién está trabajando actualmente en los locales.

Salguabot: El Asistente de Inteligencia Artificial

Puede pedirle a la IA mediante lenguaje natural que: "Añade un concierto de Rock este viernes a las 22:00", "Resta 5 botellas de Coca-Cola" o "Asigna el turno de mañana a María". La IA ejecutará las acciones en la base de datos automáticamente (Function Calling).

4. Operativas del Local

4.1 Control de Inventario y Almacén

Gestión visual del catálogo. Alternar entre el Local Principal y el Segundo Local. Genera alertas de roturas de stock bajo los límites de seguridad.

4.2 Escáner y Reconocimiento de IA

Utilizando Gemini 2.5 Flash de Google, tome una fotografía de sus facturas/albaranes y el sistema leerá los importes y nombres de proveedor y lo apuntará como un Gasto Contable de forma automática.

4.3 Agenda y Generador de Carteles

Para eventos musicales (Conciertos, Pinchadas de DJ), el sistema invocará a Imagen 3 para generar automáticamente carteles promocionales listos para publicarse en Instagram.

5. Contabilidad y Cierres

5.1 Arqueos Diarios

Introduzca el dinero contabilizado en el TPV y el cajón. El sistema contrastará el descuadre y agrupará la información para la contabilidad mensual.

5.2 Analíticas

Reportes interactivos del ratio de ingresos/gastos de sus locales y tendencias en la salud de su negocio.
